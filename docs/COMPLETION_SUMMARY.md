## 📝 演示流程

### 1. 启动服务（30 秒）
```bash
./run.sh
# 自动创建虚拟环境 + 安装依赖 + 启动
```

### 2. 多语言 PDF 生成（2 分钟）
1. 滚动到 **Document Center**
2. 选择任意产品
3. 勾选想要的语言（如 EN + CN + JP）
4. 点击 **Generate & Download PDF**
5. 浏览器自动下载

### 3. 上传自定义手册（1 分钟）
1. 准备一个 PDF/TXT 文件
2. 在 **Upload manual** 区域上传
3. 自动同步到文档中心
4. 可立即生成 PDF 或删除

### 4. 删除演示（30 秒）
1. 找到上传的文档（标记为 **Uploaded**）
2. 点击 **Delete**
3. 确认删除
4. 文档列表自动刷新
